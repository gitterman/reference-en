= bit()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Computes the value of the specified bit (bit 0 is 1, bit 1 is 2, bit 2 is 4, etc.).
[%hardbreaks]


[float]
=== Syntax
`bit(n)`


[float]
=== Parameters
`n`: the bit whose value to compute


[float]
=== Returns
The value of the bit.


[float]
=== Examples
 int n = 0x13   ;  // n = 19 = 00010011b
 n = n & bit(4) ;  // n = 16
 n = bit(5)     ;  // n = 32
